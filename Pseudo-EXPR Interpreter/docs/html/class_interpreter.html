Detailed Description

The actual Interpreter.

It combined all of the classes together to form the Interpreter. It stores in itself the instructions that actually make the code run.

Constructor & Destructor Documentation

◆ Interpreter()

Interpreter::Interpreter ( const std::string &  srcCode)

Interpreter Constructor.

Gets the source code as an arguments and creates a Tokenizer object which generates all of the tokens inside the source code. Those tokens are then parsed to a Parser object which in return parses all of the tokens and creates instructions for the Interpreter to run. All of the exceptions thrown are handled within it and spit out to the console. If an exception is throws, m_Status gets set to false

void Interpreter::interpret ( ) const

Runs the program.

Goes though all of the instructions and runs them one by one. If an instruction throws an error then interpret() ends and the error message is printed on the console

Note
