Detailed Description

A class that supports arbitrary integer length.

This class adds support of so-called BigIntegers. It allows a user to store an unsigned integer with as much length as possible. This class adds support for big integeres through storing a container of numbers with a specified radix. In this implementation the chosen radix is 100. That means that each element of the container doesn't store a bigger number than 100.

using BigInteger::RadixSize = uint16_t

This implementation uses a base of 100 while the maximum supprted value of each element is of size uint16_t.

The set up base is 100 beacuse of the overflow problem that appears when mutiplying big elements (Ex. 90 * 90)
